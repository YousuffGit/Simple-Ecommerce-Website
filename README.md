# 🛒 Simple E-Commerce Storefront

A lightweight, responsive frontend e-commerce storefront featuring a dynamic shopping cart and a sleek dark mode toggle.

**[Click Here to View the Live Demo!](https://yousuffgit.github.io/Simple-Ecommerce-Website/)**

---

## Features

* **Interactive Shopping Cart:** Add items to the cart, view the running total, and manage items in real-time.
* **Dark Mode Toggle:** Smooth, user-friendly theme switching (Light/Dark mode) that adapts to user preference.
* **Responsive Design:** Optimized for both desktop and mobile viewing.
* **No-Install Execution:** Run the entire store directly in your web browser.

---

## Tech Stack

* **HTML5:** Semantic document structure.
* **CSS3:** Custom layout styling, transitions, and dark/light theme variables.
* **JavaScript (ES6):** Dynamic DOM manipulation for cart logic and theme toggling.
